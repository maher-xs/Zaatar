# تغيير اللغة والمنطقة – Zaatar

كيف تغيّر لغة الواجهة والمنطقة الزمنية في زعتر.

## تبديل اللغة (العربية ↔ الإنجليزية)

1. افتح **الإعدادات** (Settings).
2. اذهب إلى **المنطقة واللغة** (Region & Language).
3. تحت **اللغة** (Language)، اضغط **إضافة لغة** (Add Language) وأضف **English (United States)** أو **العربية** إن لم تكن موجودة.
4. اختر اللغة المطلوبة وحرّكها لأعلى القائمة، أو اخترها كلغة العرض.

**اختصار لوحة المفاتيح:** Super+Space للتبديل بين الإنجليزية والعربية.

## تغيير المنطقة الزمنية

1. **الإعدادات** → **المنطقة واللغة**.
2. تحت **التوقيت** (Time) أو **المنطقة الزمنية**، اختر منطقتك (مثلاً دمشق، القاهرة، عمّان).

أو من الطرفية:
```bash
sudo timedatectl set-timezone Asia/Damascus
```

## لهجات عربية أخرى (مصر، الأردن، السعودية)

النظام مضبوط افتراضياً على `ar_SY` (سوريا). لتغيير الـ locale:

```bash
# مثال: مصر
localectl set-locale LANG=ar_EG.UTF-8

# مثال: الأردن
localectl set-locale LANG=ar_JO.UTF-8

# مثال: السعودية
localectl set-locale LANG=ar_SA.UTF-8
```

ثم أعد التشغيل. قد تحتاج تثبيت حزمة اللغة المناسبة:
```bash
rpm-ostree install langpacks-ar
```
