# الألعاب وتطبيقات Windows على زعتر

ثبّت التطبيقات من **Software** (Flathub): Steam، ProtonUp-Qt، Bottles، Heroic، Lutris.

## Steam
- ثبّت من Software → ابحث عن Steam
- افتح Steam وسجّل دخول
- Steam Play مفعّل — ألعاب Windows تعمل تلقائياً عبر Proton
- لأقصى توافق: افتح ProtonUp-Qt وثبّت GE-Proton

## تطبيقات Windows (.exe)
- ثبّت Bottles من Software
- افتح Bottles
- أنشئ Bottle جديد واختر نوعه (Application أو Gaming)
- اضغط Run Executable واختر ملف .exe

## ألعاب Epic Games و GOG
- ثبّت Heroic Games Launcher من Software
- افتح Heroic وسجّل دخول

## ملاحظة
ألعاب تستخدم Anti-Cheat (مثل Valorant) لا تعمل على Linux.
